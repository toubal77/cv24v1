CV24 REST HELP

La Page affiche la liste des opérations gérées par notre service REST.

URL METHODE ATTENDUE ACTION
/ ou /index GET Retourne la page d'accueil du projet
/help GET Retourne la liste des opérations gérées par le service REST
/cv24/resume/xml GET Retourne une liste résumée les CV24 présentes dans la base en flux XML avec les informations suivantes :
  • Identifiant de la CV24
  • Identité : Genre, Prénom, Nom
  • Objectif avec le statut spécifié en début de ligne
  • Diplôme le plus élevé (ou le plus récent)
En cas de succès, un flux XML conforme au schéma sera retourné. En cas d'échec, un XML de retour contenant l'identifiant et le statut de l'erreur sera retourné.
/cv24/resume GET Retourne une liste résumée les CV24 présentes dans la base en flux HTML avec les informations suivantes :
  • Identifiant de la CV24
  • Identité : Genre, Prénom, Nom
  • Objectif avec le statut spécifié en début de ligne
  • Diplôme le plus élevé (ou le plus récent)
En cas de succès, le contenu sera retourné au format HTML. En cas d'erreur, un message d'erreur HTML sera affiche
/cv24/xml/{id} GET Opération Retourne le contenu complet du CV24 dont l’identifiant est {id}

Retour Flux XML conforme au schéma XSD d'une CV24. En cas de succès, un flux XML conforme au schéma sera retourné. En cas d'échec, un XML de retour contenant l'identifiant et le statut de l'erreur sera retourné.
/cv24/html/{id} GET Opération Retourne le contenu complet du CV24 dont l’identifiant est {id} en format HTML.

En cas de succès, le contenu sera retourné au format HTML. En cas d'erreur, un message d'erreur HTML sera affiche
/cv24/insert POST Transmis Flux XML décrivant la CV24 à ajouter, conformément au schéma XSD.

En cas de succès, un flux XML conforme au schéma sera retourné. En cas d'échec, un XML de retour contenant l'identifiant et le statut de l'erreur sera retourné.
/cv24/delete/{id} DELETE Suppression du CV24 dont l’identifiant est {id} Retour Flux XML.

En cas de succès, un flux XML conforme au schéma sera retourné. En cas d'échec, un XML de retour contenant l'identifiant et le statut de l'erreur sera retourné.
